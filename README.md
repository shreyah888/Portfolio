# Shreya H Portfolio

Modern single-page developer portfolio built with React and Tailwind CSS.

## Features

- Responsive recruiter-friendly portfolio UI
- Dark modern theme
- Clean hero section with profile photo
- Skills with icons
- Experience timeline
- Project cards with image previews
- Certifications
- Resume download
- Mailto contact form
- SEO-friendly metadata

## Tech Stack

- React.js
- Vite
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

### Vercel

1. Push this project to GitHub.
2. Import the repository in Vercel.
3. Use `npm run build` as the build command.
4. Use `dist` as the output directory.

### Netlify

1. Push this project to GitHub.
2. Create a new Netlify site from the repository.
3. Use `npm run build` as the build command.
4. Use `dist` as the publish directory.

### GitHub Pages

For GitHub Pages, set a Vite `base` path in `vite.config.js` if deploying under a repository subpath.
